zorba::hexbinary::streambuf

#include <zorba/util/hexbinary_stream.h>

A hexbinary::streambuf is-a std::streambuf for encoding to and decoding from HexBinary on-the-fly. To use it, replace a stream's streambuf: 
  istream is;
  // ...
  hexbinary::streambuf hb_buf( is.rdbuf() );
  is.ios::rdbuf( &hb_buf );
Note that the hexbinary::streambuf must exist for as long as it's being used by the stream. If you are replacing the streabuf for a stream you did not create, you should set it back to the original streambuf: 
  void f( ostream &os ) {
    hexbinary::streambuf hb_buf( os.rdbuf() );
    try {
      os.ios::rdbuf( &hb_buf );
      // ...
    }
    catch ( ... ) {
      os.ios::rdbuf( hb_buf.orig_streambuf() );
      throw;
    }
    os.ios::rdbuf( hb_buf.orig_streambuf() );
  }
Alternatively, you may wish to use either , , or instead.While hexbinary::streambuf does support seeking, the positions are relative to the original byte stream.
