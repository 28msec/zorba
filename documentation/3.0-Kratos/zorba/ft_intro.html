Full Text

The Zorba processor implements the XQuery and XPath Full Text 1.0 specification.Additional documentation:
  • Full Text Stemming
  • Full Text Thesaurus
  • Full Text Tokenizer
  • Full Text Module

Unimplemented Features

The full text specification is mostly, but not completely, implemented. The features that are not (completely) implemented are:
  • The Ignore Option (bug 866924).
  • Score Variables and Using Weights Within a Scored FTContainsExpr (bug 866923).
