zorba::Stemmer

#include <zorba/stemmer.h>

A Stemmer is used to obtain the "stem" (root) word of of some word. For example the stem of "flavoring" is "flavor". A Stemmer is used by the XQuery Full Text feature.

destroy

void destroy() const =0

Destroys this Stemmer.

This function is called by Zorba when the Stemmer is no longer needed.If your StemmerProvider dynamically allocates Stemmer objects, then the implementation can simply be (and usually is) delete this.If your StemmerProvider returns a pointer to a static Stemmer object, then the implementation should do nothing.
